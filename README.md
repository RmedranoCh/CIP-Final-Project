Hi! This project is a 2D interactive clone of the classic arcade game Space Invaders, built entirely in Python using the native Tkinter graphics library. The game features a complete user experience cycle, including a main menu, real-time gameplay with an animated alien fleet, dynamic scoring, and a game-over/victory screen with fully functional "Replay" and "Exit" buttons.

Note: This game is written using standard Python Tkinter to support a robust window layout and native UI buttons locally. Since the Code in Place embedded browser console cannot open pop-up Tkinter GUI windows directly, please download the script and run it locally on your computer (via VS Code, PyCharm, or Thonny) to play the game!
